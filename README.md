# battleships-pygame
battleships remade in pygame with some extra features (group project)

# how to run
- make sure you have python and pygame
- download all .py files
- run main.py

# features
- variety of weapons earned by sinking a ship
- menu system
- slot machine and money earned from winning games, money and other data stored in a txt file (pdatahandler.py)
- MOSTLY working AI opponent with 3 different difficulties

# stuff i learned
- more object oriented programming, especially methods
- keeping program split up into different files and importing them into the main file
- menu handling loooots of menus
